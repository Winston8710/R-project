# 量販店 資料分析
R-project

##1 
建立分類模型
1. 三個月內沒有消費 - S1過客
2. 三個月內消費過一次 - N1新顧客
3. 三個月內消費一次以上，且頻率大於10次 - R1 忠實顧客
4. 三個月內消費一次以上，且頻率小於10次 - R2 一般顧客
